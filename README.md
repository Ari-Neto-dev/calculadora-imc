# Calculadora IMC

Aplicação web que calcula o Índice de Massa Corporal (IMC) a partir do peso e altura informados pelo usuário.

O projeto demonstra manipulação de DOM, validação de dados e construção de interface simples focada em experiência do usuário.

## Tecnologias utilizadas

- JavaScript
- HTML5
- CSS3

## Funcionalidades

- Inserção de peso e altura
- Cálculo automático do IMC
- Exibição da classificação do resultado
- Interface simples e responsiva

## Objetivo do projeto

Projeto desenvolvido para praticar lógica de programação, manipulação de DOM e construção de aplicações web com JavaScript.
